Generación de reportes institucionales

Esta pantalla delega en el backend el corte (30/06 o 31/12 para el reporte semestral), la clasificación y la transición de estados. La operación es asíncrona: el servidor responde 202 y la UI debe mantenerse actualizada hasta que el estado técnico se estabilice en GENERADA, APROBADA o FALLIDA.
