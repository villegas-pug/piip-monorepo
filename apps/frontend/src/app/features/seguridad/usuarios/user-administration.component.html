Administración de usuarios institucionales

Esta pantalla delega la identidad en Keycloak. PIIP nunca solicita, procesa ni almacena contraseñas. La activación inicial la realiza el propio usuario desde el correo que Keycloak le envía; la desactivación bloquea el acceso en Keycloak y PIIP conservando los registros para auditoría.
